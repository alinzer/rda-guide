This guide shows how to authenticate, submit feeds, and retrieve results using the RDA Noteworthy API.

1. ðŸ”‘ Authentication

Use your Client ID and Client Secret to obtain an access token.

curl
Python
export CLIENT_ID="your_client_id"
export CLIENT_SECRET="your_client_secret"

ACCESS_TOKEN=$(curl -s -X POST https://chemed-dev.rdanoteworthy.com/api/v1/auth/token \
  -H "Authorization: Basic $(echo -n $CLIENT_ID:$CLIENT_SECRET | base64)" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials" | jq -r '.access_token')

echo "Access Token: $ACCESS_TOKEN"
import base64
import requests

CLIENT_ID = "your_client_id"
CLIENT_SECRET = "your_client_secret"

auth_header = base64.b64encode(f"{CLIENT_ID}:{CLIENT_SECRET}".encode()).decode()

response = requests.post(
    "https://chemed-dev.rdanoteworthy.com/api/v1/auth/token",
    headers={
        "Authorization": f"Basic {auth_header}",
        "Content-Type": "application/x-www-form-urlencoded",
    },
    data={"grant_type": "client_credentials"},
)

data = response.json()
ACCESS_TOKEN = data["access_token"]
print("Access Token:", ACCESS_TOKEN)

2. ðŸ“¤ Submit a Feed

Once you have an access token, submit a text file feed for processing.

curl
Python
curl -X POST \
  "https://chemed-dev.rdanoteworthy.com/api/v1/feeds/notes/submit" \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@notes.csv;type=text/csv"
import requests

files = {"file": ("notes.txt", open("notes.csv", "rb"), "text/csv")}

response = requests.post(
    "https://chemed-dev.rdanoteworthy.com/api/v1/feeds/notes/submit",
    headers={"Authorization": f"Bearer {ACCESS_TOKEN}"},
    files=files,
)

print(response.status_code, response.text)

print(response.status_code)
print(response.json())
